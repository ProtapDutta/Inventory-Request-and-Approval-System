# Inventory Request and Approval System

A full-stack MERN application that lets employees submit inventory requests and admins approve or reject them. Built for a smooth workflow and scalable architecture.

---

## Technologies & Packages Used

### Backend
- **Node.js** – Server runtime
- **Express.js** – Web framework
- **MongoDB & Mongoose** – Database and ODM
- **dotenv** – For environment variables
- **cors** – Cross-origin resource sharing control
- **jsonwebtoken** – JWT authentication
- **bcryptjs** – Password hashing

### Frontend
- **React** (with Vite) – User Interface
- **Axios** – API requests
- **React Router** – Routing
- **Bootstrap 5** – Styling/UI components
